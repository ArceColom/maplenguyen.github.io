BASIC GIT COMMANDS

These are some basic commands in Git to familiarize yourself

Telling Git who you are


git config --global user.name "Sam Smith"
git config --global user.email sam@example.com

Creating a new local repository


git init

Adding files


git add _filename_
git add *

Committing

Committing changes to head (but not yet to the remote repository)


git commit -m "Commit message"

Committing any files you've added with git add, and also committing any files you've changed since then


git commit -a

Pushing (sending changes to the master branch of your remote repository)


git push origin master
